# 📊 HR Analytics Project – Employee Attrition Analysis

## 📌 Project Overview

This project analyzes employee attrition patterns using SQL, Python, and Power BI.
The main goal is to identify key factors affecting employee turnover and help HR teams make better decisions.

---

## 🎯 Objectives

* Analyze employee attrition trends
* Identify factors affecting employee retention
* Perform data cleaning and EDA
* Build interactive dashboard

---

## 🛠️ Tools & Technologies Used

* SQL - Data extraction and querying
* Python (Pandas) - Data cleaning
* Power BI - Dax, Data visualization and dashboard

---

## 📂 Project Structure

* HR_Analysis.ipynb
* HR_Analysis.sql
* HR_Dashboard.pbix
* README.md

---

## 📊 Key Analysis Performed

* Attrition rate calculation
* Department-wise attrition analysis
* Age and gender analysis
* Job role and salary impact
* Job satisfaction analysis

---

## 📈 Dashboard Insights

* Overall attrition rate
* Department-wise employee distribution
* Attrition by age and salary
* Key factors affecting attrition

---

## 🔍 Key Findings

* Research & Development has the highest attrition
* Low job satisfaction leads to higher attrition
* Salary and work-life balance impact retention
* Younger employees leave more
* Single employees show highest attrition compared to married employees

---

## 🚀 Conclusion

This project helps HR teams understand employee behavior and improve retention using data-driven insights.
